# 淫语翻译器！

能把中文翻译成淫语的翻译器！


## 样例

```python
import yinglish

s = '不行，那里不行。'
print(yinglish.chs2yin(s))
# 不行，那……那里不行……
```


## 接口说明

```python
def chs2yin(s, 淫乱度=0.5):
```

s是原字符串，淫乱度是0~1的实数，越大越淫乱。
